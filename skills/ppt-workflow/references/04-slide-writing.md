# 04 Slide Writing

Use this workflow to turn structure and evidence into slide titles, body copy, chart messages, speaker notes, and transitions.

## When to use

Use when the user needs page copy, sharper wording, speaker notes, chart descriptions, or a draft based on an existing structure.

## Inputs

- Goal brief.
- Slide structure.
- Data research plan.
- Existing notes, drafts, or source materials.
- Tone preference and page density.

## Process

1. Write action-oriented slide titles.
2. Make each slide support one core message.
3. Use evidence where needed and mark placeholders clearly.
4. Keep body copy concise and scannable.
5. Add chart messages before chart types.
6. Add speaker notes when the slide needs explanation beyond visible text.
7. Add transitions between chapters.
8. Flag slides that are too dense or under-supported.

## Output format

```markdown
## Slide Writing Plan
### Slide [#]: [Title]
- Core message:
- Visible copy:
  - [Bullet]
  - [Bullet]
- Chart/table suggestion:
- Speaker notes:
- Transition:
- Source/citation:
- Open issue:
```

## Handoff

Send the slide writing plan to `visual-design`. If the user needs a fast draft, send it directly to `ppt-building` with a simple style.
