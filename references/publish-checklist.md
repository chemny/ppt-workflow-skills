# Publish Checklist

Before publishing:

- [ ] `SKILL.md` exists at repository root.
- [ ] Child skills exist under `skills/*/SKILL.md`.
- [ ] `README.md` and `README.zh.md` explain installation and usage.
- [ ] `LICENSE` exists.
- [ ] No API keys, tokens, cookies, or private credentials.
- [ ] No private memory files.
- [ ] No machine-specific absolute paths.
- [ ] No `.cc-switch` synchronization logic.
- [ ] `node_modules`, generated PPTX files, screenshots, and temporary outputs are excluded.
- [ ] `tools/ppt-builder-cli` installs and passes typecheck.
- [ ] Example deck builds successfully.
- [ ] Platform compatibility is reported honestly.
